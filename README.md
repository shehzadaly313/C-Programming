# C++ Programming
This C++ program is a simple Parking Management System that manages up to 50 parking spaces. It allows the user to park five types of vehicles—Rikshaw, Car, Bus, Jeep, and Van—through a menu-driven interface that runs continuously using a while(true) loop. At the start, the program initializes variables to keep track of the total amount collected, the total number of parked vehicles, and the count of each vehicle type. It also creates a boolean array named slots with 50 elements, where false means a parking space is empty and true means it is occupied. Whenever the user selects a vehicle type, the program first checks whether the parking lot is full. If space is available, it records the current date and time using the <ctime> library, searches the slots array to find the first available parking space, marks that space as occupied, updates the parking fee and vehicle counters, and then displays a success message along with the parking time and assigned slot number. If the user chooses option 6, the program displays the day's parking records, including the total amount collected, the total number of vehicles parked, and the number of each type of vehicle. If the user chooses option 7, all records are cleared by resetting the counters to zero and marking every parking slot as empty, allowing the system to start fresh. If an invalid menu option is entered, the program informs the user and displays the menu again. Overall, this program demonstrates the practical use of variables, arrays, loops, conditional statements, counters, constants, and the C++ time library to build a basic real-world parking management application.
